This folder contains two folders - one for input .pdb files and one for output files.

INPUT:
- All individual structures
- All complex structures after alignment with 3sn6 (B2AR_Gs.pse)
- All aligned structures in one session (D1R_allGprot.pse)

OUTPUT: 
- All refined structures from HADDOCK
- A PyMol session contaning all refined structures (D1R_allGProt_Ref.pse)
