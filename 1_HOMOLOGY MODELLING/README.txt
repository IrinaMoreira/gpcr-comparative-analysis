All files contained are for homology modelling.
SEQUENCE ALIGNMENT is the folder for all sequence alignment-related information with its own README.txt file.

The scripts to be run, sequentially are:
- 1_get_homologs.py
- 2_align_sequences.py
- 3_build_models.py

Input files:
- 3sn6_A.pdb
- d1r.ali

Output files:
- d1r.B99990048.pdb (the best model)

Other necessary files which should be updated regularly:
- pdb_95.pir (database of all known non-redundant PDB structures)
