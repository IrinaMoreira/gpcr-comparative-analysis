A single folder is present with the output from the SpotOn webserver.

The input file should be a PDB file from the refinement stage.

Output files:
- HS-table.csv (all predicted Hot-Spots)
- input.pdb (the .pdb that served as input)
- NS-talbe (all predicted Null-Spots)
- run-parameters.json (running parameters for the prediction)
- SpotOn.all.csv (all calculated features and predictions)
- SpotOn.classification.csv (all classifications (HS and NS))
